# SearchApp

Приложение на `PySide6` для поиска текста в локальных и сетевых папках.

## Возможности

- поиск текста внутри файлов
- поиск без учета регистра
- поиск по локальным папкам и сетевому пути
- фильтр по расширениям файлов
- поддержка открытия найденного файла
- подготовка к сборке Windows `.exe`

## Запуск локально

```bash
pip install -r requirements.txt
python app/main.py
```
