# Сборка для Windows

На компьютере с Windows:

1. Распакуйте или откройте папку проекта.
2. Дважды нажмите `build_windows.bat`.
3. Дождитесь завершения сборки.
4. Готовое приложение будет в папке `dist\SearchApp`.
5. Для запуска откройте `SearchApp.exe`.

Если Windows спросит, чем открыть `.bat`, запускайте через `cmd`.

Если директору нужно просто пользоваться программой, передавайте ей всю папку:

`dist\SearchApp`

Внутри уже будет `SearchApp.exe`.
